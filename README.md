# Smart Driver Drowsiness Detector

A real-time driver drowsiness detection system built with Python that leverages 
Computer Vision, Fuzzy Systems, and Parallel & Distributed Computing to enhance 
road safety.

## 🚀 Features
- Real-time eye and facial landmark detection
- Fuzzy logic-based drowsiness level classification
- Parallel processing for fast and efficient detection
- Distributed computing support for scalability

## 🛠️ Technologies Used
- **Language:** Python
- **Computer Vision:** OpenCV, Dlib / MediaPipe
- **Fuzzy System:** scikit-fuzzy
- **Parallel & Distributed Computing:** multiprocessing / threading
- **Machine Learning:** NumPy, SciPy

## 📌 How It Works
1. Camera captures real-time video feed
2. Computer Vision detects facial landmarks & eye aspect ratio
3. Fuzzy logic evaluates drowsiness level
4. Alert is triggered if drowsiness is detected

## ⚙️ Installation
```bash
pip install -r requirements.txt
python main.py
```
